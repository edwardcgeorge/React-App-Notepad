# React-App-Notepad
A notepad built with Reactjs, Nodejs and Webpack

# Install
```
git clone https://github.com/edwardcgeorge/React-App-Notepad.git
cd React-App-Notepad
npm install
npm start
```
Enter this application through `http://localhost:3500`
